# Demonstrate static polymorphism (method overloading) and dynamic polymorphism (method overriding) in a single program with clear examples. [High]

```java

class Animal {

    public void makeSound() {
        System.out.println("Some generic animal sound");
    }

    
    public void info(String name) {
        System.out.println("Animal name: " + name);
    }

    public void info(String name, int age) {
        System.out.println("Animal name: " + name + ", Age: " + age);
    }
}

class Dog extends Animal {
    @Override
    public void makeSound() { // Overriding (runtime)
        System.out.println("Woof! Woof!");
    }

    public void info(String name, String breed) {
        System.out.println("Dog name: " + name + ", Breed: " + breed);
    }
}

public class Main {
    public static void main(String[] args) {

        Animal animal = new Animal();
        animal.info("Generic Animal");
        animal.info("Generic Animal", 5);

        Dog dog = new Dog();
        dog.info("Buddy");
        dog.info("Buddy", 3);
        dog.info("Buddy", "Golden Retriever"); // Overloaded in subclass

        Animal myDog = new Dog(); // Reference of parent, object of child
        myDog.makeSound(); // Calls Dog's overridden method at runtime
    }
}


```


## Output:
 
```java

Animal name: Generic Animal
Animal name: Generic Animal, Age: 5
Animal name: Buddy
Animal name: Buddy, Age: 3
Dog name: Buddy, Breed: Golden Retriever
Woof! Woof!

```
