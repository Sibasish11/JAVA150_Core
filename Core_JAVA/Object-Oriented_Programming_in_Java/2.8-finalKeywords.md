# Write a program to demonstrate final keywords with variables, methods, and classes. Explain their purpose. [Medium]

```java

class Constants {
    public static final double PI = 3.14159; // constant value
}

// Final method example
class Vehicle {
    public final void startEngine() {
        System.out.println("Engine starting...");
    }
}

// Attempting to override a final method will cause a compile-time error
class Car extends Vehicle {
    // Uncommenting this will cause an error:
    // public void startEngine() { System.out.println("Overridden!"); }
}

final class Utility {
    public void showMessage() {
        System.out.println("This is a utility class.");
    }
}


public class Main {
    public static void main(String[] args) {
        // Using a final variable
        System.out.println("Value of PI: " + Constants.PI);

        // Using a final method
        Car myCar = new Car();
        myCar.startEngine();

        // Using a final class
        Utility util = new Utility();
        util.showMessage();
    }
}

```

## Output:

```java

Value of PI: 3.14159
Engine starting...
This is a utility class.

```
