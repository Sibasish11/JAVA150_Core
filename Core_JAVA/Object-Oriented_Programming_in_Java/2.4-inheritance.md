# Implement a class hierarchy: Vehicle -> TwoWheeler -> Motorcycle. Add appropriate methods and fields to each, demonstrating inheritance. [Medium]

```java

class Vehicle {
    String brand;
    int year;

    Vehicle(String brand, int year) {
        this.brand = brand;
        this.year = year;
    }

    void start() {
        System.out.println(brand + " vehicle starting...");
    }

    void stop() {
        System.out.println(brand + " vehicle stopped.");
    }
}

class TwoWheeler extends Vehicle {
    boolean hasCarrier;

    TwoWheeler(String brand, int year, boolean hasCarrier) {
        super(brand, year);
        this.hasCarrier = hasCarrier;
    }

    void displayInfo() {
        System.out.println("Brand: " + brand + ", Year: " + year +
                           ", Has Carrier: " + hasCarrier);
    }
}

class Motorcycle extends TwoWheeler {
    int engineCapacity; // in cc

    Motorcycle(String brand, int year, boolean hasCarrier, int engineCapacity) {
        super(brand, year, hasCarrier);
        this.engineCapacity = engineCapacity;
    }

    void revEngine() {
        System.out.println(brand + " motorcycle revving engine (" + engineCapacity + "cc)!");
    }
}

public class Main {
    public static void main(String[] args) {
        Motorcycle m = new Motorcycle("Yamaha", 2023, false, 155);

        m.start();        // From Vehicle
        m.displayInfo();  // From TwoWheeler
        m.revEngine();    // From Motorcycle
        m.stop();         // From Vehicle
    }
}


```

### Output:

```java

Yamaha vehicle starting...
Brand: Yamaha, Year: 2023, Has Carrier: false
Yamaha motorcycle revving engine (155cc)!
Yamaha vehicle stopped.

```
