# Implement a singleton design pattern for a Logger class, ensuring only one instance of the class can be created. [High]

```java

class Logger {
    // Step 1: Create a private static instance 
    private static Logger instance = new Logger();

    // Step 2: Make the constructor private to prevent direct instantiation
    private Logger() {
        System.out.println("Logger instance created.");
    }

    // Step 3: Provide a public static method to get the instance
    public static Logger getInstance() {
        return instance;
    }

    public void log(String message) {
        System.out.println("[LOG]: " + message);
    }
}

public class Main {
    public static void main(String[] args) {
        // Both references point to the same instance
        Logger logger1 = Logger.getInstance();
        Logger logger2 = Logger.getInstance();

        logger1.log("Application started.");
        logger2.log("Logging another message.");

        System.out.println("logger1 == logger2 ? " + (logger1 == logger2));
    }
}


```

# Output:

```java

Logger instance created.
[LOG]: Application started.
[LOG]: Logging another message.
logger1 == logger2 ? true

```
