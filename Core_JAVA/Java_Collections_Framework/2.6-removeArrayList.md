# Write a program to remove all occurrences of a specific element from an ArrayList. [Medium]

```java

import java.util.ArrayList;

public class RemoveOccurrences {
    public static void main(String[] args) {
        ArrayList<Integer> numbers = new ArrayList<>();
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);
        numbers.add(20);
        numbers.add(40);
        numbers.add(20);

        System.out.println("Original List: " + numbers);

        int target = 20;  // element to remove

        // ✅ Method 1: Using removeIf (Java 8+)
        numbers.removeIf(n -> n == target);
        System.out.println("After removing " + target + " using removeIf: " + numbers);

        // Reset list again for demonstration
        numbers.clear();
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);
        numbers.add(20);
        numbers.add(40);
        numbers.add(20);

        // ✅ Method 2: Using removeAll
        numbers.removeAll(java.util.Collections.singleton(target));
        System.out.println("After removing " + target + " using removeAll: " + numbers);

        // Reset list again for demonstration
        numbers.clear();
        numbers.add(10);
        numbers.add(20);
        numbers.add(30);
        numbers.add(20);
        numbers.add(40);
        numbers.add(20);

        // ✅ Method 3: Using while loop
        while (numbers.contains(target)) {
            numbers.remove(Integer.valueOf(target)); // remove by object, not index
        }
        System.out.println("After removing " + target + " using while loop: " + numbers);
    }
}


```

### Output:

```java

Original List: [10, 20, 30, 20, 40, 20]
After removing 20 using removeIf: [10, 30, 40]
After removing 20 using removeAll: [10, 30, 40]
After removing 20 using while loop: [10, 30, 40]
```
