# Write a program to find the frequency of each word in a given text using a HashMap. [High]

```java

import java.util.HashMap;
import java.util.Map;

public class WordFrequency {
    public static void main(String[] args) {
        
        String text = "This is a simple text and this text is simple";

        // Convert text to lowercase and split into words (using space as delimiter)
        String[] words = text.toLowerCase().split("\\s+");

        // Create a HashMap to store word frequencies
        HashMap<String, Integer> wordCount = new HashMap<>();

        // frequency of each word
        for (String word : words) {
            // If word already exists, increment count, else add it with count = 1
            wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
        }

        // Print word frequencies
        System.out.println("Word Frequencies:");
        for (Map.Entry<String, Integer> entry : wordCount.entrySet()) {
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }
    }
}


```

## Output:

```java

Word Frequencies:
this -> 2
is -> 2
a -> 1
simple -> 2
text -> 2
and -> 1


```
