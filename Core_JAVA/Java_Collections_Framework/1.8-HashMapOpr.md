# Write a program to demonstrate common HashMap operations: put, get, containsKey, remove, size. [Medium]

```java

import java.util.HashMap;

public class HashMapOperations {
    public static void main(String[] args) {
        // Create a HashMap (key: Integer, value: String)
        HashMap<Integer, String> map = new HashMap<>();

        map.put(1, "Apple");
        map.put(2, "Banana");
        map.put(3, "Mango");
        System.out.println("After put operations: " + map);

        System.out.println("Value for key 2: " + map.get(2));

 
        System.out.println("Contains key 3? " + map.containsKey(3));
        System.out.println("Contains key 5? " + map.containsKey(5));

        map.remove(1);
        System.out.println("After removing key 1: " + map);

        // 5. size() - Get number of entries
        System.out.println("Size of map: " + map.size());
    }
}


```

### Output:

```java

After put operations: {1=Apple, 2=Banana, 3=Mango}
Value for key 2: Banana
Contains key 3? true
Contains key 5? false
After removing key 1: {2=Banana, 3=Mango}
Size of map: 2

```
