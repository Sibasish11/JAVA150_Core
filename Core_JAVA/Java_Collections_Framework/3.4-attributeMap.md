# Write a program to group objects by an attribute using a Map (e.g., group Student objects by major). [High]

```java

import java.util.*;

class Student {
    private String name;
    private String major;

    public Student(String name, String major) {
        this.name = name;
        this.major = major;
    }

    public String getName() {
        return name;
    }

    public String getMajor() {
        return major;
    }

    @Override
    public String toString() {
        return name;
    }
}

public class GroupByMajor {
    public static void main(String[] args) {

        List<Student> students = Arrays.asList(
                new Student("Alice", "Computer Science"),
                new Student("Bob", "Mathematics"),
                new Student("Charlie", "Computer Science"),
                new Student("David", "Physics"),
                new Student("Eva", "Mathematics"),
                new Student("Frank", "Physics")
        );


        Map<String, List<Student>> groupedByMajor = new HashMap<>();

        for (Student s : students) {
            // If major not in map, add new list
            groupedByMajor.putIfAbsent(s.getMajor(), new ArrayList<>());
            // Add student to respective major group
            groupedByMajor.get(s.getMajor()).add(s);
        }

        for (Map.Entry<String, List<Student>> entry : groupedByMajor.entrySet()) {
            System.out.println("Major: " + entry.getKey() + " -> Students: " + entry.getValue());
        }
    }
}


```

## Output:

```java

Major: Computer Science -> Students: [Alice, Charlie]
Major: Mathematics -> Students: [Bob, Eva]
Major: Physics -> Students: [David, Frank]

```
