# Write a program to demonstrate common ArrayList operations: add, remove, get, size, contains, indexOf. [Low]

```java

import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        // Creating an ArrayList of Strings
        ArrayList<String> fruits = new ArrayList<>();

        fruits.add("Apple");
        fruits.add("Banana");
        fruits.add("Orange");
        fruits.add("Mango");
        System.out.println("After adding: " + fruits);

        fruits.remove("Banana"); // remove by value
        fruits.remove(1);        // remove by index (removes "Orange")
        System.out.println("After removing: " + fruits);

        String firstFruit = fruits.get(0);
        System.out.println("Element at index 0: " + firstFruit);

        // 4. Size of the ArrayList
        System.out.println("Size: " + fruits.size());

        System.out.println("Contains Mango? " + fruits.contains("Mango"));
        System.out.println("Contains Banana? " + fruits.contains("Banana"));

        int index = fruits.indexOf("Mango");
        System.out.println("Index of Mango: " + index);
    }
}


```

### Output:

```java

After adding: [Apple, Banana, Orange, Mango]
After removing: [Apple, Mango]
Element at index 0: Apple
Size: 2
Contains Mango? true
Contains Banana? false
Index of Mango: 1


```
