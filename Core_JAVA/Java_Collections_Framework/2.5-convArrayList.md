# Write a program to convert an ArrayList to an array and vice-versa. [Low]

```java

import java.util.ArrayList;
import java.util.Arrays;

public class ArrayListArrayConversion {
    public static void main(String[] args) {
        //Converting ArrayList to Array
        ArrayList<String> list = new ArrayList<>();
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");

        System.out.println("ArrayList: " + list);

        // toArray() method
        String[] array = list.toArray(new String[0]);
        System.out.println("Converted Array: " + Arrays.toString(array));

        // âœ… Convert Array to ArrayList
        String[] fruits = {"Mango", "Orange", "Pineapple"};

        // Arrays.asList() creates a fixed-size list, so wrap it in a new ArrayList for flexibility
        ArrayList<String> newList = new ArrayList<>(Arrays.asList(fruits));
        System.out.println("Converted ArrayList: " + newList);
    }
}


```

### Output:

```java

ArrayList: [Apple, Banana, Cherry]
Converted Array: [Apple, Banana, Cherry]
Converted ArrayList: [Mango, Orange, Pineapple]
```
