# Write a program to find the frequency of each word in a given text using a HashMap. [High]

```java

import java.util.HashMap;
import java.util.Map;

public class WordFrequency {
    public static void main(String[] args) {
        String text = "Java is simple and Java is powerful and Java is popular";

        // Convert text to lowercase & split into words (ignoring punctuation)
        String[] words = text.toLowerCase().split("\\s+");

        // HashMap to store word frequencies
        HashMap<String, Integer> wordCount = new HashMap<>();

        // Count frequencies
        for (String word : words) {
            word = word.replaceAll("[^a-z]", ""); // remove punctuation
            if (!word.isEmpty()) {
                wordCount.put(word, wordCount.getOrDefault(word, 0) + 1);
            }
        }

        // Print word frequencies
        System.out.println("Word Frequencies:");
        for (Map.Entry<String, Integer> entry : wordCount.entrySet()) {
            System.out.println(entry.getKey() + " -> " + entry.getValue());
        }
    }
}


```

## Output:

```java

Word Frequencies:
java -> 3
is -> 3
simple -> 1
and -> 2
powerful -> 1
popular -> 1


```
