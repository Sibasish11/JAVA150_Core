# Compare and contrast HashSet and LinkedHashSet and TreeSet. Demonstrate their behavior with a few elements. [Medium]

```java

/*
 Compare and Contrast HashSet, LinkedHashSet, and TreeSet

1. Internal Implementation:
   - HashSet:
       → Uses HashMap internally.
       → Elements are stored based on hash codes.
       → No guaranteed order of elements.
   - LinkedHashSet:
       → Uses LinkedHashMap internally.
       → Maintains insertion order of elements.
   - TreeSet:
       → Implements NavigableSet (uses Red-Black Tree internally).
       → Elements are stored in sorted (ascending) order.

2. Performance (Average Case):
   - HashSet:
       → add(), remove(), contains() = O(1)
   - LinkedHashSet:
       → Similar to HashSet (O(1)), but slightly slower due to maintaining order.
   - TreeSet:
       → add(), remove(), contains() = O(log n) due to tree structure.

3. Null Elements:
   - HashSet: Allows one null element.
   - LinkedHashSet: Allows one null element.
   - TreeSet: Does NOT allow null (throws NullPointerException).

4. When to Use:
   - HashSet: Best for quick lookups without caring about order.
   - LinkedHashSet: When insertion order must be preserved.
   - TreeSet: When sorted order is required.
*/

import java.util.*;

public class SetComparisonDemo {
    public static void main(String[] args) {

        Set<String> hashSet = new HashSet<>();
        hashSet.add("Banana");
        hashSet.add("Apple");
        hashSet.add("Mango");
        hashSet.add("Apple"); // duplicate

        Set<String> linkedHashSet = new LinkedHashSet<>();
        linkedHashSet.add("Banana");
        linkedHashSet.add("Apple");
        linkedHashSet.add("Mango");
        linkedHashSet.add("Apple"); // duplicate

      
        Set<String> treeSet = new TreeSet<>();
        treeSet.add("Banana");
        treeSet.add("Apple");
        treeSet.add("Mango");
        treeSet.add("Apple"); // duplicate

        System.out.println("HashSet:       " + hashSet);
        System.out.println("LinkedHashSet: " + linkedHashSet);
        System.out.println("TreeSet:       " + treeSet);
    }
}


```

## Output:

```java

HashSet:       [Apple, Banana, Mango]   // Order not guaranteed
LinkedHashSet: [Banana, Apple, Mango]   // Preserves insertion order
TreeSet:       [Apple, Banana, Mango]   // Sorted order


```
