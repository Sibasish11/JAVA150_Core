# Write a program to demonstrate common HashSet operations: add, remove, contains, size. [Low]

```java

import java.util.HashSet;

public class HashSetOperations {
    public static void main(String[] args) {
        // Create a HashSet of integers
        HashSet<Integer> set = new HashSet<>();

        // 1. add() → Adding elements
        set.add(10);
        set.add(20);
        set.add(30);
        set.add(40);
        set.add(20); // duplicate, will not be added

        System.out.println("Initial HashSet: " + set);

        // 2. contains() → Check if element exists
        System.out.println("Contains 30? " + set.contains(30));
        System.out.println("Contains 50? " + set.contains(50));

        // 3. remove() → Remove an element
        set.remove(20);
        System.out.println("After removing 20: " + set);

        // 4. size() → Number of elements
        System.out.println("Size of HashSet: " + set.size());

        // 5. isEmpty() → Check if empty
        System.out.println("Is HashSet empty? " + set.isEmpty());

        // 6. clear() → Remove all elements
        set.clear();
        System.out.println("After clearing: " + set);
    }
}


```

# Output:

```java

Initial HashSet: [40, 20, 10, 30]
Contains 30? true
Contains 50? false
After removing 20: [40, 10, 30]
Size of HashSet: 3
Is HashSet empty? false
After clearing: []


```
