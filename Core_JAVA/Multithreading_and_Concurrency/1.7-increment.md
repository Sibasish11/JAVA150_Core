# Solve the race condition problem (Q106) using the synchronized block. [Medium]

```java


class Counter {
    private int count = 0;

    public void increment() {
        // only this block is synchronized (not the whole method)
        synchronized (this) {
            count++;
        }
    }

    public int getCount() {
        return count;
    }
}

class WorkerThread extends Thread {
    private Counter counter;

    public WorkerThread(Counter counter) {
        this.counter = counter;
    }

    @Override
    public void run() {
        for (int i = 0; i < 1000; i++) {
            counter.increment();
        }
    }
}

public class SyncBlockExample {
    public static void main(String[] args) {
        Counter counter = new Counter();

        WorkerThread t1 = new WorkerThread(counter);
        WorkerThread t2 = new WorkerThread(counter);
        WorkerThread t3 = new WorkerThread(counter);

        t1.start();
        t2.start();
        t3.start();

        try {
            t1.join();
            t2.join();
            t3.join();
        } catch (InterruptedException e) {
            e.printStackTrace();
        }

        System.out.println("Final counter value (expected 3000): " + counter.getCount());
    }
}


```

## Expected Output:
```
Final counter value (expected 3000): 3000
```
