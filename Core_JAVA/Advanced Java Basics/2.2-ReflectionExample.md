# Write a program that uses Java Reflection to inspect a class at runtime. Get its methods, fields, and constructors. [High]

```java

import java.lang.reflect.*;

// A sample class to inspect
class Person {
    private String name;
    private int age;

    // Constructor
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // Methods
    public String getName() { return name; }
    public int getAge() { return age; }
    private void secretMethod() { System.out.println("This is private!"); }
}

public class ReflectionExample {
    public static void main(String[] args) {
        try {
            // Step 1: Get the Class object
            Class<?> cls = Person.class;

            // Step 2: Print class name
            System.out.println("Class Name: " + cls.getName());

            // Step 3: Get Constructors
            System.out.println("\nConstructors:");
            Constructor<?>[] constructors = cls.getConstructors();
            for (Constructor<?> c : constructors) {
                System.out.println(" " + c);
            }

            // Step 4: Get Fields
            System.out.println("\nFields:");
            Field[] fields = cls.getDeclaredFields();  // includes private fields
            for (Field f : fields) {
                System.out.println(" " + f);
            }

            // Step 5: Get Methods
            System.out.println("\nMethods:");
            Method[] methods = cls.getDeclaredMethods(); // includes private methods
            for (Method m : methods) {
                System.out.println(" " + m);
            }

        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}

```
