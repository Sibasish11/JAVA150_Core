# Implement a program to read and parse comma-separated values (CSV) from a file. [High]

```java

import java.io.BufferedReader;
import java.io.FileReader;
import java.io.IOException;

public class CSVReaderExample {
    public static void main(String[] args) {
        String filePath = "data.csv"; // Change to your CSV file path

        try (BufferedReader br = new BufferedReader(new FileReader(filePath))) {
            String line;
            
            System.out.println("Reading CSV file and parsing values:\n");

            while ((line = br.readLine()) != null) {
                // Split line by comma
                String[] values = line.split(",");

                // Print values for each row
                for (String value : values) {
                    System.out.print(value.trim() + " | "); // trim removes spaces
                }
                System.out.println(); 
            }

        } catch (IOException e) {
            System.err.println("Error reading the file: " + e.getMessage());
        }
    }
}


```

## ✅ Example CSV file (data.csv):
```
ID,Name,Age,Department
1,Alice,22,Computer Science
2,Bob,23,Electronics
3,Charlie,21,Mechanical
```
## ✅ Output:
- Reading CSV file and parsing values:

```
ID | Name | Age | Department | 
1 | Alice | 22 | Computer Science | 
2 | Bob | 23 | Electronics | 
3 | Charlie | 21 | Mechanical |
```
